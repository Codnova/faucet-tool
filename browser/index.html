HNS Address Generator

HNS Multisig Address Generator

This tool generates the first receiving address, in the first account of a BIP44 HD wallet. It also generates the 24 word BIP39 mnemonic phrase which was used to seed the wallet. It is important that you keep this seed phrase private. It is the only way to access your wallet. Do not share your seed phrase with anyone that should not have access to the contents of the wallet.

This tool also generates the specific keypair used to derive the receiving address. The keypair could instead be included in a multisig address.

This tool generates an m-of-n multisig address. It also generates the redeem script necessary to spend coins sent to the address. It is important that you record this redeem script. You must include the redeem script in a transaction spending these coins.
